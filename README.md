# wizardry5rewrite
```
現代版への総論として
可能な限りABで行い、遷移ページ数を減らす。
情報の少ないものについて、情報を提示する。
キャンプ：
　パーティ欄は七行はいる。先頭がパーティ
商店：
　所持しているか装備しているかを金額の末尾に記述する。
  装備可能かはパーティ欄のステータス項目に表示すれば良い。もっている、装備している
＊ロングソード　　　　　　　３０００００＠０１
１弓　　　　　　　　　　　　　　４０００＠９９
２石ころ　　　　　　　　　　　　　　１０＠０１

装備：
　道具の使用と魔法の使用は同じ
　キャンプ時にはYで、LRで道具を魔法。カーソルは記憶
マップ表示：
　Xで常に固定する。
　十字キーで座標移動。下部に座標とNPCなどがあれば表示する。
　LRで階層の変更。階層は上部。マップ移動のカーソルはブリンクして透ける。
 --------B00 階層名---------
|　 　　　　　　　　　　　　　|
|　　　＊　　　　　　　　　　 |
|　　　　　　　　　　　　　　 |
 ----X00Y00 情報表示--------
```
